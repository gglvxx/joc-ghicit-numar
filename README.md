# 🎮 Joc Ghicit Număr

Acesta este un proiect interactiv creat în **Python**, folosind biblioteca **Streamlit**.

## 🚀 Joacă online
Poți testa jocul direct în browser aici: 
👉 https://joc-ghicit-numar-andrei.streamlit.app/

## 🛠️ Tehnologii folosite
* **Python**
* **Streamlit** (pentru interfața web)

## 📝 Cum funcționează?
1. Calculatorul alege un număr secret între 1 și 100.
2. Tu introduci un număr în căsuța de text.
3. Aplicația îți oferă indicii ("Prea mic" / "Prea mare") până când ghicești!
